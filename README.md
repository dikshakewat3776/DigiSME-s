# DigiSME-s
Business owner uploads a video of his utilities that are meant for trade and using Object detection technique the object is detected and price summary is displayed . If customer searches for the product which matches with business utilities then the nearest location business video is recommended with the contact number.



# Stakeholder:  Company

Company Registration (prerequisite to company location and other details)
Video/ images upload feature for company. 
Object detection of commodities. (freshness/expiry bound)
Extraction of commodity list and price tagging by the company. (Editable by the company.)
SME uploaded.


# Stakeholder: User

User searches for a product.
Display company based on nearest available outlet / freshest commodity.
Recommend
